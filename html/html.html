HTML Boilerplate

The first tag in every html document is an HTML doctype. This tells you which version of html the browser should interpret the document as.

the tag for HTML5 is <!DOCTYPE html>

this is a self closing tag

The next tag that encompases the entire html document is: <html lang="en-US">

lang is the attribute and sets the browser to interpret the text as english with a US dialect. Possibly this is for the visually impaired or any text to speech feature

Within the html tag there is a <head> and a <body> tag, these are sibling tags at the same level but within the <html> tags
